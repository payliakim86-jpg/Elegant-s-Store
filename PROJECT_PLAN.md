# 🛍️ Elegant's Store - План розробки фронтенду

## Обзор проекту
**Назва:** Elegant's Store  
**Тип:** E-commerce магазин одягу  
**Мета:** Створити елегантний, сучасний сайт для продажу жіночого та чоловічого одягу  
**Стек:** Next.js 14 + TypeScript + Tailwind CSS + React  

---

## 📱 Структура сторінок

### 1. **Головна сторінка (Home / Index)**
- Hero секція з логотипом Elegant's Store
- Слайдер популярних колекцій
- Секція "Нові надходження"
- Секція "Найпопулярніші товари"
- Секція з категоріями одягу
- Testimoniails / Відгуки клієнтів
- Newsletter підписка
- Call-to-action кнопки

### 2. **Каталог товарів (Products / Shop)**
- Сітка товарів з фільтрацією:
  - За категоріями (рубашки, платья, штани, куртки тощо)
  - За розміром
  - За кольором
  - За ціною (слайдер)
  - За статтю (жіночий, чоловічий, унісекс)
- Сортування (популярність, ціна, новизна)
- Пагінація або infinite scroll
- Карточка товару з основною інформацією

### 3. **Сторінка товару (Product Detail)**
- Великі зображення товару (галерея з можливістю зуму)
- Основна інформація:
  - Назва
  - Ціна
  - Рейтинг та кількість відгуків
  - Опис
  - Матеріал, склад
- Вибір розміру та кольору
- Кількість товару
- Кнопка "Додати в кошик"
- Кнопка "Вподобати" (для улюблених)
- Відгуки та рейтинги
- Рекомендовані товари (related products)

### 4. **Кошик (Shopping Cart)**
- Список добавлених товарів
- Зображення, назва, ціна за одиницю
- Можливість змінити кількість
- Кнопка видалення товару
- Підрахунок загальної вартості
- Рекомендовані товари для додання
- Кнопка "Перейти до оформлення"
- Кнопка "Продовжити покупки"

### 5. **Оформлення замовлення (Checkout)**
- Поступовий процес (multi-step form):
  - **Крок 1:** Доставка (адреса)
  - **Крок 2:** Спосіб доставки
  - **Крок 3:** Спосіб оплати
  - **Крок 4:** Перевірка замовлення
- Сумарна вартість з деталізацією:
  - Сума товарів
  - Вартість доставки
  - Податки
  - Загальна вартість
- Промокод / купон (поле для введення)

### 6. **Сторінка користувача (Account / Profile)**
- Інформація профілю
- Мої замовлення (історія замовлень)
- Мої адреси доставки
- Мої улюблені товари (wishlist)
- Налаштування акаунту
- Вихід з акаунту

### 7. **Аутентифікація**
- **Сторінка входу (Login)**
  - Email/Password поля
  - Кнопка "Забув пароль"
  - Посилання на реєстрацію
  - Соціальний вхід (Google, Facebook - опціонально)

- **Сторінка реєстрації (Sign Up)**
  - Форма з полями (ім'я, email, пароль, підтвердження пароля)
  - Согласие на умови використання
  - Кнопка реєстрації

- **Сторінка відновлення пароля (Reset Password)**

### 8. **Сторінка "Про нас" (About)**
- Історія бренду Elegant's Store
- Місія та цінності
- Команда (фото, посадовці)
- Соціальні мережі

### 9. **Сторінка контактів (Contact)**
- Форма зв'язку
- Email, телефон, адреса
- Карта (Карти Google)
- Соціальні мережі

### 10. **FAQ / Допомога (Help Center)**
- Часті запитання про:
  - Доставку
  - Повернення товарів
  - Розміри
  - Матеріали
  - Способи оплати
- Форма для зв'язку з підтримкою

### 11. **Сторінка Категорії (Category)**
- Схожа на каталог, але для конкретної категорії
- Фільтри по підкатегоріям

### 12. **Сторінка пошуку (Search Results)**
- Результати пошуку за запитом
- Фільтри та сортування
- Інформація про кількість знайдених результатів

### 13. **Сторінка 404 (Not Found)**
- Дружелюбне повідомлення про помилку
- Кнопки для навігації (на головну, каталог)

### 14. **Сторінка успішного замовлення (Order Confirmation)**
- Номер замовлення
- Деталі замовлення
- Адреса доставки
- Посилання на відстеження
- Email з підтвердженням
- Рекомендовані товари

## 🎨 Компоненти (Components)

### Навігація
- `Header` - Топ навігація з логотипом, меню, пошуком, кошиком, профілем
- `Navigation` - Основне меню категорій
- `Footer` - Футер з посиланнями, підпискою на новини, контактами

- `ProductCard` - Карточка товару в сітці

- `CartItem` - Елемент в кошику
- `CartSummary` - Підсумок кошика
- `CheckoutForm` - Форма оформлення
- `StepIndicator` - Індикатор кроків

### Форми
- `LoginForm` - Форма входу
- `RegisterForm` - Форма реєстрації
- `ContactForm` - Форма контактів
- `NewsletterForm` - Форма підписки

### Утиліти
- `Modal` - Модальне вікно
- `Toast` - Сповіщення
- `Pagination` - Пагінація
- `Rating` - Рейтинг зірок
- `Button` - Кнопка (різні варіанти)
- `Input` - Текстове поле
- `Dropdown` - Випадаючий список

## 🛠️ Технологічний стек

- **Framework:** Next.js 14 (App Router)
- **Мова:** TypeScript
- **Стилізація:** Tailwind CSS
- **State Management:** Context API + useReducer або Zustand
- **HTTP Client:** Fetch API або Axios
- **Форми:** React Hook Form + Zod для валідації
- **Іконки:** Lucide React або React Icons
- **Карусель:** Swiper.js
- **Модальні вікна:** Headless UI
- **Тостинги:** Sonner або React Toastify
- **Анімації:** Framer Motion (опціонально)

## 🔌 API Endpoints (очікувані)

### Товари
- `GET /api/products` - Список всіх товарів (з фільтрацією)
- `GET /api/products/:id` - Деталі товару
- `GET /api/categories` - Список категорій
- `GET /api/products/search?q=query` - Пошук товарів

### Користувачі
- `POST /api/auth/login` - Вхід
- `POST /api/auth/register` - Реєстрація
- `POST /api/auth/logout` - Вихід
- `GET /api/users/profile` - Профіль користувача
- `PUT /api/users/profile` - Оновлення профілю

### Замовлення
- `POST /api/orders` - Створення замовлення
- `GET /api/orders` - Список замовлень користувача
- `GET /api/orders/:id` - Деталі замовлення

### Кошик
- `GET /api/cart` - Отримати кошик
- `POST /api/cart/items` - Додати товар
- `PUT /api/cart/items/:id` - Оновити товар
- `DELETE /api/cart/items/:id` - Видалити товар
